# 🔮 AI 관상 프로그램 (CAM) 주요 특징 및 기술적 배경

## 📜 프로젝트 소개

이 프로그램은 Python의 **`Tkinter`**를 GUI로, **`OpenCV`**를 이미지 처리 엔진으로 활용하여 웹캠(CAM) 기반의 **실시간 관상 분석**을 시도하는 오락 및 학습 목적의 애플리케이션입니다.

사용자의 웹캠 피드를 화면에 실시간으로 표시하며, **"관상 분석"** 버튼 클릭 시 현재 얼굴을 캡처하여 주요 특징점(눈, 코, 입 등)을 감지하고, 이에 기반한 **아주 간단한 분석 결과**를 제공합니다.

> ⚠️ **주의사항:** 본 프로그램의 관상 분석 로직은 과학적 근거가 없는 단순화된 예시이며, 실제 관상학적 의미나 정확성을 보장하지 않습니다. 재미와 기술 학습 목적으로만 사용해 주세요.
